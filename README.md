# Python User Input Script

## 📌 Overview
This is a beginner Python script that collects user input and performs basic logic.

The program:
- Asks for the user’s name
- Greets the user
- Asks for their age
- Calculates and displays their age next year

## 💻 Code Example
```python
name = input("What's your name? ")
print("Welcome,", name)

age = input("How old are you? ")
print("Next year you'll be", int(age) + 1)
```

## ▶️ How to Run
1. Open terminal / command prompt
2. Navigate to the file location
3. Run:

```
python hello.py
```
